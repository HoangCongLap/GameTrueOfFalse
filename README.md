# 1.Yêu cầu:

- Nếu đúng 10 lần liên tiếp sẽ hiện chữ "**Win**", ngược lại hiện chữ "**Lose**", lúc này thời gian chương trình sẽ dừng lại. 
- Thời gian đếm ngược của mỗi ván là 3 giây, khi hết thời gian bạn sẽ thua và hiện chữ "**Lose**".
